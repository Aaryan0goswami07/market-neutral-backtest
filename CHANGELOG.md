# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2025-02-20

### Added
- Initial release of Market Neutral Backtest Framework
- Z-score based signal generation
- Market neutrality validation (beta tracking)
- Transaction cost modeling (configurable bps)
- 14+ institutional-grade performance metrics
- Interactive Streamlit web interface
- CSV export functionality
- Comprehensive documentation
- Sample data and examples

### Features
- Real-time parameter adjustment
- Visual analytics (equity curve, drawdown, exposure)
- Multiple preset pairs (AAPL/MSFT, JPM/BAC, XOM/CVX)
- Custom ticker input
- Market neutrality dashboard
- Performance attribution

### Documentation
- Complete README with examples
- Installation instructions
- Usage guide
- Methodology explanation
- Contributing guidelines
